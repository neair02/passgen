# Руководство пользователя — PassVault

## Запуск приложения

```bash
python app.py
```
Откройте **http://127.0.0.1:5000** в браузере.

---

## Генерация пароля

1. В блоке **Генератор** настройте параметры:
   - **Длина** — переместите ползунок (4–64 символа)
   - **A–Z** — включить заглавные буквы
   - **0–9** — включить цифры
   - **!@#…** — включить спецсимволы
2. Нажмите **Сгенерировать**.
3. Пароль появится в зелёном поле. Нажмите 📋 чтобы скопировать.

---

## Сохранение пароля

1. После генерации заполните поля:
   - **Сайт / сервис** — обязательно (например: `GitHub`)
   - **Логин / email** — необязательно
2. Нажмите **💾 Сохранить пароль**.
3. Запись появится в таблице **Хранилище** ниже.

---

## Просмотр и копирование

В таблице хранилища:
- Нажмите **👁** рядом с паролем, чтобы показать/скрыть его.
- Нажмите **📋** чтобы скопировать пароль в буфер обмена.

---

## Поиск

Введите слово в поле поиска (в блоке Хранилище) и нажмите **🔍**.  
Поиск работает по названию сайта и логину.  
Нажмите **✕** чтобы сбросить поиск.

---

## Редактирование

1. Нажмите **✏️** напротив нужной записи.
2. Измените данные. Для генерации нового пароля нажмите 🔄.
3. Нажмите **💾 Сохранить**.

---

## Удаление

Нажмите **🗑** напротив записи и подтвердите удаление.
